# 🛍️ Store-X

Store-X is a full-stack e-commerce web application built with a modern tech stack. It allows users to browse products, add items to a cart, and complete a checkout flow with a clean and responsive UI.

This project demonstrates full-stack development skills including frontend UI/UX, backend API design, and database integration.

---

## 🚀 Live Demo

👉 https://store-x-djme.onrender.com/

---

## ✨ Features

- 🛒 Add to Cart functionality (Demo login required)
- 🔐 Demo Login for quick access (no signup needed)
- 📦 Checkout system with order summary
- 🔎 Product browsing and category filtering
- ⚡ Fast and responsive UI
- 🔗 RESTful API integration
- 🗄️ Persistent data with PostgreSQL

---

## 🧪 Demo Usage

To explore the app:

1. Click **"Demo Login"** in the navbar  
2. Browse products  
3. Add items to cart  
4. Proceed to checkout  
5. Place an order  

⚠️ Note: This is a demo project. No real payments are processed.
